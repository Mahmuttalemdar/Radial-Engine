Radial-Engine
=============

3D Game engine(C++, OpenGL) with an editor(Qt).

- Skinned meshes
- Texture splatting
- Terrain editing
- Animation
- Physics(Bullet)
- GPU based procedural content generation(perlin)
- Procedural terrain collider for physics
- FBX asset files, auto sync
- Entity inheritance on data level
- JSON data format
